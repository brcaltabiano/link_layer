### Passos para execucao

1. Abrir um terminal localizado na pasta raiz deste projeto

2. Executar "make all"

3. Executar "make run"

4. A execução do programa deve iniciar após isso. Um prompt irá aparecer pedindo
ao usuário que digite a mensagem a ser transmitida pelo programa.
